# Predict & Prevent DDoS Attacks

## 📌 Overview
This project simulates a system that predicts and prevents Distributed Denial of Service (DDoS) attacks using traffic pattern analysis. It combines detection logic with preventive mechanisms to ensure system security.

## 🎯 Objectives
- Detect abnormal traffic patterns
- Predict potential DDoS attacks
- Trigger preventive actions automatically

## 🧩 Features
- Dataset-based analysis
- Rule-based prediction model
- Automated prevention actions
- Modular architecture

## 🛠 Tech Stack
- Python

## 📂 Project Structure
- main.py → Execution
- model.py → Prediction logic
- data_loader.py → Dataset processing
- prevention.py → Response system
- config.py → Settings
- dataset.csv → Sample data
- test_system.py → Unit testing

## ⚙️ How It Works
1. Load traffic dataset
2. Train prediction model
3. Analyze incoming traffic
4. Detect potential attacks
5. Trigger prevention mechanisms

## ▶️ How to Run
python main.py

## 🧪 Testing
python test_system.py
